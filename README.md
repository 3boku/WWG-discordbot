# WWG-discordbot
> Women Who Go에서 발표하는 dis'go'd 소스코드 레포지스토리 입니다

![go-image] 1.20.6 <br>

"ping"을 입력하면 "pong"을 반환하는 간단한 디스코드 Bot입니다.
"discordgo"패키지를 활용하여 개발하였습니다.

## 디렉토리

```
📦Go-To-Daegu
 ┣ 📜 go.mod - 프로젝트 패키지 관리자
 ┣ 📜 main.go
 ┗ 📜 README.md - 프로젝트의 README 파일
```

## Install
```sh
go get -u github.com/bwmarrin/discordgo
```

## Using
```shell
go run main.go
```
